# Elenco istruzioni visualizzate:

Per inizializzare una repository ho usato:
```bash
git init
```

Il secondo comando che abbiamo visto, per visualizzare lo status della repository Ã¨:
```bash
git status
```

Il terzo comando che abbiamo visto serve per aggiungere elementi alla **staging area**:
```bash
git add .
```

Il quarto comando che abbiamo visto serve per effettuare un commit:
```bash
git commit -m "Aggiunto..."
```

___

Per clonare una repo, mi basta avere il LINK ed esegure:
```bash
git clone LINK
```

Per scaricare le modifiche di una repo, mi basta usare:
```bash
git pull
```
